Methods for filtering

We cover implementations for common use cases. One can fall back to the low level OpenCV API for more flexibility if needed.

cr.vision.filters.filter.gaussian_blur(image, kernel_size=5, sigma=0)[source]¶

Filters an image using Gaussian filter of a given kernel size where we assume that the filter is symmetric.
