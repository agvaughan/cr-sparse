cr.sparse.lop.LinearOperator¶

class cr.sparse.lop.LinearOperator(times: Callable[[jax._src.numpy.lax_numpy.ndarray], jax._src.numpy.lax_numpy.ndarray], trans: Callable[[jax._src.numpy.lax_numpy.ndarray], jax._src.numpy.lax_numpy.ndarray], m: int, n: int)[source]¶

Represents a finite linear operator \(T : A -> B\) where \(A\) and \(B\) are finite vector spaces.

Parameters

  • times – A function implementing \(T(x)\)

  • trans – A function implementing \(T^H (x)\)

  • m – The dimension of the destination vector space \(B\)

  • n – The dimension of the source vector space \(A\)
