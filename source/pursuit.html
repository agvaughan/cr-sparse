Sparse Recovery Algorithms¶

Data Types

RecoverySolution

Represents the solution of a sparse recovery problem

Utilities

abs_max_idx(h)

Returns the index of entry with highest magnitude

gram_chol_update(L, v)

Incrementally updates the Cholesky factorization \(G = L L^T\) where \(G = \Phi^T \Phi\)

Greedy pursuit based Algorithms¶

Orthogonal Matching Pursuit

solve(Phi, y, max_iters[, max_res_norm])

Solves the recovery/approximation problem \(y = \Phi x + e\) using Orthogonal Matching Pursuit

solve_multi(Phi, y, max_iters[, max_res_norm])

Vectorized version of solve.

Compressive Sampling Matching Pursuit

solve(Phi, y, K[, max_iters, res_norm_rtol])

Solves the sparse recovery problem \(y = \Phi x + e\) using Compressive Sampling Matching Pursuit

Iterative Hard Thresholding

solve(Phi, y, K[, step_size, max_iters, …])

Solves the sparse recovery problem \(y = \Phi x + e\) using Iterative Hard Thresholding
